# wenv

**W**ith **env** file

# Install

```bash
$ deno task install
```

# Usage

```
$ wenv -h
wenv - Run a command with dotenv file
USAGE
    wenv [OPTIONS] [args]...

OPTIONS
    -f, --file <string>    dotenv file
    --allow-empty          allow empty values
    -V, --version          Prints version information
    -h, --help             Prints help information

ARGS
    [args]...    
```
